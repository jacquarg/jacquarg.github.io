Politique de confidentialité

L'outils Proxy Redirect n'analyse et ne conserve aucunes données. Vos données personnelles seront traité uniquement dans votre navigateur, donc sur vos appareils.

Pour vous en assurer, vous pouver consulter le code ici. La distribution fidèle de ce programme sur la page http://jacquarg.github.io/proxy_redirect/ étant garantie par le système github.io .
